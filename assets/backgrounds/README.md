# 背景图 / 头像图 贡献包

本目录为 **Xuqi LLM Chat** 项目贡献的参考背景图与头像图资源。

> 项目地址：[https://github.com/Frischar/Xuqi_LLM](https://github.com/Frischar/Xuqi_LLM)

---

## 目录说明

| 文件夹 | 用途 |
|--------|------|
| `backgrounds/` | 聊天页面背景图，供用户在设置页选择使用 |
| `avatars/` | 角色头像 / 立绘参考图，供用户在角色卡中引用 |

---

## 贡献内容

### 背景图（`backgrounds/`）

提供多张高质量背景图，涵盖以下风格：

| 文件名 | 风格描述 | 推荐场景 |
|--------|----------|----------|
| `*.png / *.jpg / *.webp` | 二次元 / 插画风 | 角色扮演聊天界面 |
|  |  |  |

- **分辨率**：建议 1920×1080 及以上
- **格式**：PNG、JPG、WebP 均支持
- **文件大小**：单张不超过 30 MB（项目上传限制）

### 头像图（`avatars/`）

提供角色头像参考图，适用于：

- 角色卡（Character Card）头像
- 差分立绘（Sprite）参考
- 用户个人头像

---

## 使用方式

### 背景图

1. 进入 Xuqi LLM WebUI → **设置页（/config）**
2. 在背景图选项中上传本目录中的图片
3. 或通过 URL 方式引用本地图床

### 头像图

1. 进入 **角色卡页（/config/card）**
2. 导入角色卡时引用本目录中的图片作为头像
3. 或在立绘管理页（/config/sprite）中上传差分图
